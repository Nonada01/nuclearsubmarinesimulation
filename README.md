# 핵추진 잠수함 공간좌표 시뮬레이터

**공간 좌표와 도형**을 주제로 한 3D 잠수함 시뮬레이션 교육 툴입니다.

## 주요 기능

### 📐 공간 좌표 시각화
- **3D 좌표계**: X, Y, Z 축 실시간 표시
- **실시간 좌표값**: 위치(x, y, z)와 회전각(pitch, yaw, roll) 표시
- **라디안/도 변환**: 각도를 라디안과 도 단위로 동시 표시
- **위치 마커**: 현재 위치를 구체로 표시하고 바닥까지 수직선 연결

### 🔷 기하학적 도형 구성
잠수함은 다음 기본 도형들로 구성됩니다:
- **원기둥** (CylinderGeometry): 주선체
- **원뿔** (ConeGeometry): 함수, 함미
- **직육면체** (BoxGeometry): 함교탑, 수평타, 수직타

### 📊 벡터 시각화
- **전진 방향 벡터** (파란색): Z축 방향
- **상승 방향 벡터** (초록색): Y축 방향  
- **우현 방향 벡터** (빨간색): X축 방향
- 모든 벡터는 회전 변환에 따라 실시간 업데이트

### 🎯 추가 기능
- **궤적 추적**: 이동 경로를 3D 공간에 노란색 선으로 표시
- **와이어프레임 모드**: 도형의 구조를 명확히 볼 수 있는 모드
- **그리드 평면**: 공간 인식을 돕는 격자 바닥
- **자유 카메라**: 마우스로 회전/확대/이동 가능

## 제어 방법

### 시뮬레이션 제어
- **추진력**: 전후진 속도 조절 (-5 ~ +5)
- **Pitch**: 상하 각도 제어 (승강타)
- **Yaw**: 좌우 방향 제어 (방향타)
- **Roll**: 좌우 회전 제어

### 시각화 옵션
- 와이어프레임 / 솔리드 모드 전환
- 방향 벡터 ON/OFF
- 좌표축 표시 ON/OFF
- 그리드 표시 ON/OFF

- **React Three Fiber**: React용 Three.js 래퍼

## 교육적 활용

이 시뮬레이터는 다음 개념들을 학습하는 데 유용합니다:

1. **3D 좌표계**: 직교 좌표계에서의 위치 표현
2. **회전 변환**: Euler 각(pitch, yaw, roll)을 이용한 3D 회전
3. **벡터**: 방향과 크기를 가진 물리량 표현
4. **기본 도형**: 원기둥, 원뿔, 구, 직육면체의 3D 표현
5. **좌표 변환**: 로컬 좌표계에서 월드 좌표계로의 변환

## 게임/작업물 개발 활용

- **좌표계 이해**: 3D 게임 개발 시 필수적인 좌표 시스템 학습
- **물리 시뮬레이션**: 6자유도 운동 구현 참고
- **벡터 연산**: 방향 계산, 회전 변환 등의 실제 구현 예시
- **UI/UX 레퍼런스**: 실시간 텔레메트리 표시 방법
